# 專案

一個擁有所有運算資訊的容器

## 大綱

爲了管理專案, 前往專案頁面

<img style="width:50%" src="./../../../static/project/ProjectPage.png" />

在這個頁面, 你可以透過工具列完成所有動作

* 新增專案
* 匯出專案
* 其他...

<img style="width:100%" src="./../../../static/project/ProjectPage2.png" />

你可以按 UUID 快速進入流程管理頁面

<img style="width:100%" src="./../../../static/project/ProjectPage3.png" />

## 新增專案

讓我們嘗試新增專案

<img style="width:100%" src="./../../../static/project/NewProject0.png" />
<img style="width:50%" src="./../../../static/project/NewProject.png" />

你必須輸入名稱, 否則不會接受

<img style="width:50%" src="./../../../static/project/NewProject2.png" />

接下來, 我們可以選擇流程清單樣板 <br />
如果你沒有選擇, 將會新增空白專案容器

<img style="width:50%" src="./../../../static/project/NewProject3.png" />

這是樣板選擇清單, 你可以打開群組然後選擇你要的

<img style="width:50%" src="./../../../static/project/NewProject4.png" />

樣板區文字會顯示你選擇的樣板

<img style="width:50%" src="./../../../static/project/NewProject5.png" />

你也可以在這裡選擇綁定資料庫

<img style="width:50%" src="./../../../static/project/NewProject6.png" />

完成後按 "建立"
